# CloseButton — bouton de fermeture

Référence exécutable : `CloseButton.dc.html` · Présent dans tous les modals.

## Dimensions
40×40px · radius 12 · icône X 17px stroke 2.4.

## Couleurs
Fond `gray-75`, icône `gray-600` · hover fond `gray-100` · press scale(.96).

## Position
Coin haut-droit du modal, aligné sur le titre (rangée flex, titre flex:1).

## Accessibilité
`aria-label="Fermer"`. Cible 40px (tolérée car > 36px et isolée ; ne pas réduire).

## Règle
Toujours ce composant — jamais un X en texte ou une icône nue.
