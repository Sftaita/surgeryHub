# StatusPill & Badges

## StatusPill
Pill 24–26px, padding 0 10–11px, 12px/700. Paires exactes (voir `design-tokens.json → semantic.status`) :
Proposée (vert), Confirmée (vert), En cours (vert + point 7px `green-500` pulsant 1.6s), À encoder / En attente (ambre), À venir (bleu), Refusée (gris).
Sur photo/vert foncé : fond `rgba(20,77,56,.9)` texte blanc.

## Badge compteur (nav / sidebar)
Pill `red-600`, texte blanc 10.5–11.5/800, min-width 17–21px, bord 2px blanc, positionné coin haut-droit de l'icône (top −6, right −11). Contenu = nb d'offres proposées ; masqué à 0.

## Tags
Pill 24px 12px/600–700 : type (fond `green-100` texte `green-800`), spécialité (outline `gray-200` texte `gray-600`), « Nouveau » (20px, `green-100`/`green-800` 11/700), « À préciser » (`amber-50`/`amber-700`).

## Règle
Couleur + libellé, toujours. Aucune pastille sans texte.
